Senior Capstone

This was a program created for Western Digital as part of my senior project. I worked with three other students on this semester long project. My role was team lead, which consisted of delegating tasks within the team and communicating with the representatives at Western Digital. Successfully negotiated the scope of the project and provided a dashboard that collected/displayed information about their laboratory computers.

The Discord bot is designed to be able to be able to serve multiple groups and multiple courses for each group. Users are able to send commands to request updates on specific classes using the course ID number. When the bot recieves the update list, it will check if users are requesting any of the classes in the list and send a notification.

Networking Project

This program was created to notify users when a class they have been trying to sign up for has an open seat. There are two parts to the program, the web scraper and the Discord bot. The web scraper reads information about every class offered that semester and keeps track of how many seats are available for each class. If the number of seats increased or decreased it would add it to the update list.

The Discord bot is designed to be able to be able to serve multiple groups and multiple courses for each group. Users are able to send commands to request updates on specific classes using the course ID number. When the bot recieves the update list, it will check if users are requesting any of the classes in the list and send a notification.
